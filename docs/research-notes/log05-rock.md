# ロックにおける主体と作品の確定不能性  
——存在ではなく、成立と作動としてのリアリティ

ロックにおいて、主体と作品は原理的に確定しにくい。主体は「この人が語っている」という単一の中心に収斂せず、録音・編集・反復再生・共同性・メディア像といった複数の回路の交点として立ち上がる。作品もまた、譜面のように同一の対象として保存されるというより、テイク選択、ミックス、リミックス／リマスター、ライブでの改変、カバーや引用によって、境界がたえず更新される成立条件の束として現れる。ここで確定しているのは「正体」ではなく、成立の仕方と作動の連鎖である。

この確定不能性がもたらすのは、存在論的問いの不発である。すなわち「ロックとは何か」「この曲の本当の意味は何か」「この人の本心は何か」と問うほど、答えは固定された実体としては現れず、むしろ「どのように成立し、どのように作用し、どのように反復され、どのように変形されるか」へと問いが移動する。ロックのリアリティは、対象の同一性（これがそれだ）ではなく、強度の配分、媒介の操作、反復がつくる経験の持続、共同性の組織といった「存在の仕方」に宿る。

もっとも、ここには強い反論がある。第一に、ロックはしばしば「本物」「リアル」「魂」といった語彙で語られ、スターの人格やオーセンティシティをめぐる争いが絶えない。これは、ロックがむしろ存在論的欲望を最も強く刺激するジャンルである、という反論である。第二に、録音物は物理的・デジタル的に固定された対象なのだから、作品はむしろ確定している、という反論もありうる。第三に、ライブでの共同性や振る舞いが重視されるほど、主体のカリスマ化（実体化）が進むのではないか、という指摘も成り立つ。

しかし、これらの反論はロックの性格を否定するというより、ロックが孕む二重性を示す。ロックが「本物」を語りたくさせるのは、まさに主体と作品が確定しにくいからである。確定しないものに対して、人は本質を置きたくなる。ロックの本物主義は、ロックの本質であるというより、確定不能性を埋め合わせるために繰り返し発生する回収の欲望である。しかもその欲望は、しばしば空振りする。人格を実体化しても、録音や編集や場の変化がその実体を攪乱する。作品を固定しても、別テイク、別ミックス、別の再生環境が同一性を揺らす。つまり「本物」をめぐる存在論は、ロックにおいて成立するが、同時にロックによって絶えず破られる。

録音物が固定対象である、という反論も同様に部分的に正しい。しかし固定されているのはデータであって、作品の同一性ではない。録音は、出来事を保存するだけでなく、切断・選別・強度の調整によって出来事を別の現実として再構成する。さらに再生のたびに、環境・身体・共同性の条件が異なるかぎり、「同じ音源」が同じ作品として経験されるとは限らない。ロックにおける作品は、固定された対象である以前に、反復のなかで作動し直す条件体系である。

以上を踏まえると、ロックの重要性は「存在を語れない神秘」としての否定神学にあるのではなく、否定神学へ回収される誘惑を生みつつも、それを完遂させない点にある。ロックは、主体と作品を実体化したくさせるが、同時にそれを攪乱し、問いを存在（何であるか）から存在の仕方（どう成立し、どう作用するか）へ押し戻す。その押し戻しこそがロックの批評性であり、人を「固定された正体」ではなく「作動する様態」へ向かわせる力である。
